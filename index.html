🌱 Sensor de Humedad

Este sensor mide la cantidad de agua en la tierra. Las plantas necesitan un equilibrio entre humedad y sequedad para mantenerse sanas.

Nuestra flor avisa si tiene sed o si hay exceso de agua, optimizando el riego y evitando pudrición.

☀️ Sensor de Luz

Usando una fotocélula (LDR), medimos cuánta luz recibe la planta.

La flor detecta si está en sombra o luz intensa y puede sugerir ser movida a un lugar mejor.

💡 Razón del Proyecto

Este proyecto une tecnología y naturaleza para facilitar el cuidado de plantas. Ayuda a prevenir errores comunes y crea un vínculo más consciente con el entorno vegetal.
